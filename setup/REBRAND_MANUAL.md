# note プロフィール リブランド手順書

自動化スクリプト `setup/profile_update.py` が動かない場合の手動手順。
note 側の設定画面は頻繁にHTMLが変わるため、セレクタベースの自動化が
失敗することがある。その場合は以下の手順で手動更新する。

## 変更内容

| 項目 | 現在 | 変更後 |
|---|---|---|
| 表示名 | Ryoka｜NV CLOUD公式 | 出張買取DX研究所｜NV CLOUD公式 |
| プロフィール文 | （現状） | 下記「プロフィール文案」から選択 |

## 手動更新手順

### 1. note にログイン

1. <https://note.com/login> を開く
2. NV CLOUD の note 用メール/Google アカウントでログイン

### 2. アカウント設定画面へ

1. 右上のアイコンをクリック
2. メニューから「アカウント設定」を選択
   - または直接 <https://note.com/settings/account> を開く

### 3. 表示名を変更

1. 「名前」欄を探す（通常ページ上部）
2. 現在の「Ryoka｜NV CLOUD公式」を全選択して削除
3. 新名称を入力:
   ```
   出張買取DX研究所｜NV CLOUD公式
   ```
4. （可能なら）自己紹介欄に下記プロフィール文案から1つを貼り付け
5. ページ下部の「保存」または「更新」ボタンをクリック
6. 「保存しました」などの通知が出れば完了

### 4. 確認

1. <https://note.com/kaitori_nv_cloud> （公開プロフィール）を開く
2. 表示名が「出張買取DX研究所｜NV CLOUD公式」になっているか確認
3. ブラウザキャッシュで古い表示が残っている場合は Shift + リロード

## プロフィール文案

業界キーワード（出張買取／古物商／DX）、便益、LP誘導を含む短文バージョン。

### 案A（業界課題 → 解決のトーン、2文）

> 出張買取・古物商業界のDX推進を研究・発信するNV CLOUD公式note。査定・在庫・集客をひとつにまとめるSaaS「NV CLOUD」で、1人1日5件を超える現場でも回せるオペレーションを提案します。詳細は公式サイトへ → https://nvcloud.jp

### 案B（ビジョン提示 → 便益、1文強め）

> 「出張買取をもっと儲かる仕事に」——古物商向けSaaS NV CLOUD が運営するDX研究所。査定業務・LINE集客・在庫管理をまとめて効率化し、現場の生産性を2倍にする実践知をお届けします。サービス詳細 → https://nvcloud.jp

どちらも120〜160字程度で、note プロフィール欄（だいたい300字まで）に収まる想定。

## トラブルシューティング

### 「名前」欄が見つからない

- note の設定UIが刷新されている可能性あり
- `アカウント設定`ではなく、`プロフィール設定`の中に移動しているケースあり
- 左サイドメニューから「プロフィール」や「表示設定」を確認

### 保存ボタンが押せない（グレーアウト）

- 必須項目が空になっていないか確認
- 表示名に使えない文字（特殊記号など）が入っていないか確認
- `｜`（全角パイプ）は使用可、`|`（半角パイプ）でも可

### 変更が反映されない

- ログアウト → 再ログイン
- ブラウザのキャッシュをクリア
- 別ブラウザ（シークレットモード）で <https://note.com/kaitori_nv_cloud> を開いて確認

## 自動化スクリプト再実行

セレクタ問題でなく一時的な失敗だった場合は、再実行で通ることがある。

```bash
cd /Users/apple/NorthValueAsset/note-pipeline
uv run python setup/profile_update.py \
    --display-name "出張買取DX研究所｜NV CLOUD公式" \
    --bio "出張買取・古物商業界のDX推進を研究・発信するNV CLOUD公式note。査定・在庫・集客をひとつにまとめるSaaS「NV CLOUD」で、1人1日5件を超える現場でも回せるオペレーションを提案します。詳細は公式サイトへ → https://nvcloud.jp"
```

失敗時のスクリーンショットは `setup/logs/profile_*.png` に保存される。
