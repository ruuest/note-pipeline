## 🎯 ブランドポジショニング

**NV CLOUD** は**出張買取業者向けのSaaS**。2系統のオーディエンスに発信する:

| 軸 | 対象 | シェア | 主目的 |
|---|---|---|---|
| **BtoB（主）** | 買取業者・店舗運営者・FC加盟店 | 70% | 無料体験/資料DLへの送客 |
| **BtoC（従）** | 家財処分したい消費者・遺品整理検討層 | 30% | 「こういうSaaSを使う業者は信頼できる」という業者間接送客 |

**コア・プロミス**: 「現場の工数を減らし、数字で経営を見える化するSaaS」

**差別化モート（内部認識のみ、発信しない）**: データモート / スイッチングコスト / ネットワーク効果。

---

## 📣 3つの発信軸

各投稿は以下3軸のいずれかに明確に属する。複数軸の混在は避ける（1投稿1メッセージ）。

### 軸A: ペイン訴求

**意図**: 現場の痛みを言語化→共感→NV CLOUDが解決する予感。

**典型トピック**:
- 査定工数が多すぎる（1件30-60分→SaaSで10分）
- ダブルブッキング問題
- 写真撮影疲れ（同じアングルを毎回撮る苦痛）
- 紙台帳の検索性ゼロ問題
- 会計連携の手打ち地獄
- 相場把握の属人化
- 在庫滞留の可視化不足

**推奨比率**: 全投稿の **40%**

### 軸B: 機能紹介

**意図**: 具体的にNV CLOUDのどの機能がどう解決するか。スクショ/動画併用推奨。

**典型トピック**:
- AI査定補助機能
- スマホ撮影→自動在庫登録
- 会計ソフト連携（freee/マネフォ）
- 古物台帳の自動記載
- 出張スケジュール衝突検知
- 相場データベース
- FC加盟店別レポート

**推奨比率**: 全投稿の **35%**

### 軸C: お客様の声 / 実績

**意図**: 数値とストーリーで「使うとこう変わる」を具体化。

**典型トピック**:
- 導入前後の工数比較（30%削減等）
- 月次処理件数のBefore/After
- 担当者の声（匿名OK）
- 機能追加のきっかけエピソード
- 業界イベント/セミナー登壇報告

**推奨比率**: 全投稿の **25%**

## 🔇 絶対NGワード（発信禁止）

以下は投稿前に自動スキャン＋強制ブロック。置換案はトーンガイド+compliance_rules.yaml参照。

| NG語 | 理由 | 置換方向 |
|---|---|---|
| **粗利 / 原価** | 競合/顧客の内部情報に踏み込む、下品 | 「工数」「処理件数」「見える化」に置換 |
| **掛け率** | 買取価格の秘匿事項、発信は業界マナー違反 | 「適正価格判定」「相場参照」に置換 |
| **仕入れテク / 集客ノウハウ** | ハウツー系インフルエンサー文脈、NV CLOUDのブランドに合わない | SaaS機能の話に置換 |
| **月粗利1000万 / 月商XXX万確実** | 月粗利1000万超は内部情報、外部発信禁止（memoryに明記） | 数字を出す場合は「工数30%削減」「処理件数2倍」等の業務指標に限定 |
| **必ず / 確実に / 絶対 / 100%** | 景表法CR001/CR002 | 「多くの場合」「事例として」「傾向として」 |
| **業界最安 / No.1 / 唯一** | 景表法CR003/CR004/CR005 | 「相場より効率的」「高い評価」「独自の」 |
| **誰でも / 簡単に稼げる** | 景表法CR007（成功保証暗示） | 「多くの業者が取り組める」 |
| **リスクゼロ / 元本保証** | 景表法CR008/CR009 | 使用禁止 |
| **煽り系**（今すぐ！急げ！限定！） | ブランド毀損 | 数字+事実で説得 |

**技術実装**: `compliance_rules.yaml` の CR001-CR015（景表法15件）をX投稿パイプラインで全件評価。

## 🎨 トーン仕様

### 文体

- **基本**: です・ます調 / 断定は避け「〜のようです」「〜と考えます」
- **敬語レベル**: 丁寧だが固すぎない（ビジネス雑誌の記事程度）
- **一人称**: 「当社」/「NV CLOUD」/「私たち」（「俺」「僕」NG）
- **二人称**: 「皆様」/「業者様」/「ご担当者様」（「あなた」は文脈次第で可）

### 構成原則

| 原則 | 具体 |
|---|---|
| **数字で示す** | 「時間がかかる」→「1件45分→10分に短縮」 |
| **煽らない** | 感嘆符「！」は1ツイート1個まで / 疑問符「？」は意図的フック時のみ |
| **具体的に** | 「便利」→「在庫一覧から伝票を3クリックで発行」 |
| **業界用語は丁寧に** | 古物台帳、行商届出等は初出で簡単な補足 |
| **ネガティブを断定しない** | 「古臭い」「ダメ」→「課題がある」「改善余地がある」 |

### 絵文字ルール

**使用可能（全投稿で最大2個まで）**:
- 📊 数値データ / KPI
- 📈 成長・改善
- 📉 削減
- 🔥 新機能告知（月1-2回まで）
- ✅ チェックリスト項目
- 🏢 企業・店舗
- 💼 業務・経営
- 📱 モバイル・アプリ画面
- 🎯 目標・KPI

**使用禁止**:
- 🎉🎊（誇張祝祭感）
- 💰💸💴（拝金イメージ）
- 🚀（ベンチャー感が過剰）
- 😊😎🙌（馴れ馴れしい）
- 🤑💎（成金イメージ）
- ❤️（ブランドに合わない）

### ハッシュタグ

- **1投稿に2-3個まで**（compliance_rules.yaml CR046違反回避）
- **推奨固定タグ**: `#買取業界` / `#出張買取` / `#古物商` / `#SaaS`
- **トレンドタグ乱用禁止**: `#バズれ` `#拡散希望` `#フォロー返し` 全てNG

## 🔗 CTA仕様

### CTAパターン

| パターン | 使用頻度 | 文例 |
|---|---|---|
| **無料体験** | 高 | 「機能詳細+無料体験はこちら → https://nvcloud-lp.pages.dev/」 |
| **資料DL** | 中 | 「資料ダウンロードはLPから → https://nvcloud-lp.pages.dev/」 |
| **お問い合わせ** | 低 | 「導入相談・お見積りはDMまたはLPから」 |

### URLの扱い

- **必ず**: `https://nvcloud-lp.pages.dev/` のみ使用（他URL全てNG）
- **UTMパラメータ推奨**: `?utm_source=x&utm_medium=thread&utm_campaign=[topic]`
- **短縮URL**: 現状なし、将来 `nvcloud.jp/go/` 自社短縮を導入予定
- **CTAツイート**: スレッド最終ツイートに1つだけ、本文冒頭ツイートには入れない（Xのアルゴでリンクあり投稿はリーチ30-50%減）

### プロフィール誘導パターン（アルゴ回避）

```
1本目（フックツイート、リンクなし）
↓ ぶら下げ返信で続き
2-6本目（本編、リンクなし）
↓ ぶら下げ返信
最終ツイート（CTA + リンク1本）
```

## 🧪 品質チェックリスト（投稿前）

投稿前にAIが以下を全件セルフチェック、1件でもNGなら再生成:

- [ ] NGワード50件（compliance_rules.yaml）に該当しない
- [ ] 各ツイート本文が135字以内（URL込みで140字制限）
- [ ] 絵文字が1投稿2個以内
- [ ] 感嘆符が1ツイート1個以内
- [ ] ハッシュタグが1投稿3個以内
- [ ] スレッド全体が3-7ツイート
- [ ] CTAは最終ツイート1箇所のみ
- [ ] URL は https://nvcloud-lp.pages.dev/ のみ
- [ ] 月粗利1000万等の内部情報を含まない
- [ ] 景表法CR001-CR015 全件クリア
- [ ] ステマ規制CR026-CR035 の依頼関係が該当する場合は #PR 明記
- [ ] 数字を出す場合は根拠・条件が明示されている
